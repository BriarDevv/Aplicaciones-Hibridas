Maletín API

API RESTful para replicar el inventario tipo "attaché case" de Resident Evil 4. Gestiona usuarios, catálogo de items y el estado del maletín de cada jugador.

Usuarios

MétodoRutaDescripción
POST/api/users/registerRegistrar un nuevo usuario
POST/api/users/loginIniciar sesión (devuelve JWT)
GET/api/usersObtener todos los usuarios
GET/api/users/:idObtener un usuario por ID
DELETE/api/users/:idEliminar un usuario
PATCH/api/users/:id/nameActualizar nombre
PATCH/api/users/:id/emailActualizar email
PATCH/api/users/:id/passwordActualizar contraseña

Catálogo de items

MétodoRutaDescripción
GET/api/itemsListar items (filtro opcional ?cat=weapon)
GET/api/items/:idObtener un item por itemId
POST/api/itemsCrear un item nuevo en el catálogo
PATCH/api/items/:idEditar un item
DELETE/api/items/:idBorrar un item del catálogo

Maletín del usuario

MétodoRutaDescripción
GET/api/briefcaseObtener el maletín del usuario logueado
PUT/api/briefcaseReemplazar todo el estado del maletín
PATCH/api/briefcase/sizeCambiar el tamaño del maletín
DELETE/api/briefcaseVaciar el maletín
POST/api/briefcase/itemsColocar un item en una posición
PATCH/api/briefcase/items/:idxMover o rotar la pieza en el índice idx
DELETE/api/briefcase/items/:idxSacar la pieza del índice idx
